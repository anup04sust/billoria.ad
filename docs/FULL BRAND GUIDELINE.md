# 📘 BILLORIA ADPOINT — FULL BRAND GUIDELINE

---

# 1. 🧠 Brand Foundation

## ✅ Brand Name

**BILLORIA ADPOINT**

---

## 🎯 Tagline

**Primary:**
**“Clarity in Every Adpoint.”**

**Secondary (product use):**
“Discover. Compare. Book Outdoor.”

---

## 🧭 Brand Mission

To digitize and organize Bangladesh’s fragmented outdoor advertising market into a clear, accessible, and actionable platform.

---

## 🌍 Vision

To become the **default infrastructure layer** for outdoor advertising across Bangladesh.

---

## 💎 Core Value Proposition

* Brands → **Clarity + Control**
* Owners → **Visibility + Demand**
* Market → **Structure**

---

# 2. 🎙️ Brand Voice & Tone

## 🧩 Personality

* Calm authority
* Structured
* Product-focused
* Intelligent (not flashy)

---

## 🗣️ Tone Rules

### ✅ DO:

* Be clear and direct
* Use action-driven language
* Sound like a platform, not an agency

### ❌ DON’T:

* Overhype (“best”, “ultimate”, “world-class”)
* Use vague marketing fluff
* Sound like a service vendor

---

## 💬 Example Messaging

| Use Case | Copy                                            |
| -------- | ----------------------------------------------- |
| Hero     | “Explore billboard adpoints across Bangladesh.” |
| CTA      | “Browse Adpoints”                               |
| Product  | “Compare locations before you decide.”          |
| Value    | “Turn scattered options into clear decisions.”  |

---

# 3. 🎨 Visual Identity

---

## 🎯 Logo Concept

### Core Idea:

**Adpoint = Selected location in a structured network**

---

## 🧩 Logo Structure

### Primary Mark:

* 🔴 Red circle (Adpoint)
* ⚪ White billboard shape (minimal, perspective)
* Clean geometric layout

---

## 📏 Logo Rules

### Clear Space

* Minimum padding: **1× circle radius**

### Minimum Size

* Digital: **24px height**
* Print: **10mm**

---

## ❌ Don’t:

* Add gradients
* Add shadows
* Distort proportions
* Add outlines

---

# 4. 🎨 Color System

## 🎯 Primary Colors

| Color          | Hex     | Usage                 |
| -------------- | ------- | --------------------- |
| 🔴 Adpoint Red | #E11D48 | Primary brand, logo   |
| ⚪ White        | #FFFFFF | Background / contrast |
| ⚫ Dark Base    | #0F172A | UI / text background  |

---

## 🎯 Secondary Colors

| Color      | Hex     | Usage             |
| ---------- | ------- | ----------------- |
| Slate Gray | #64748B | Secondary text    |
| Light Gray | #E2E8F0 | Borders / UI      |
| Soft Red   | #FB7185 | Hover / highlight |

---

## ⚠️ Rules

* Red = highlight only (don’t overuse)
* Keep UI mostly dark + white + gray
* Maintain high contrast

---

# 5. 🔤 Typography

## 🎯 Primary Font

* **Inter** (recommended)
* Alternative: **Roboto / SF Pro**

---

## 🧩 Font Style

| Usage   | Style           |
| ------- | --------------- |
| Heading | Bold / SemiBold |
| Body    | Regular         |
| UI      | Medium          |

---

## 📏 Scale

* H1: 32–40px
* H2: 24–28px
* Body: 14–16px
* Small: 12px

---

# 6. 🧩 Iconography

## 🎯 Style

* Minimal
* Solid shapes
* No stroke-heavy icons

---

## 📌 Common Icons

* Billboard
* Map / location
* Road / network
* Filter / grid

---

## ❌ Avoid

* Emoji-style icons
* 3D icons
* Over-detailed illustrations

# 7. 🖥️ UI / Product Design System

---

## 🎯 Core Principles

### 1. Clarity First

* Every screen answers: *“What can I do next?”*

---

### 2. Structured Layout

* Grid-based UI
* Clean spacing
* No clutter

---

### 3. Data-Driven Feel

* Tables
* Filters
* Map + list combination

---

## 🧩 Key UI Elements

### Buttons

* Primary: Red background, white text
* Secondary: Outline / gray

---

### Cards (Adpoints)

* Image
* Location
* Size/type
* Availability

---

### Filters

* City
* Road type
* Billboard type
* Price range
