# 🔄 3) Onboarding Flow (Simple but smart)

---

## 🧠 A. Agency Onboarding Flow

### Step 1: Signup

* Name
* Company
* Role
* Phone

### Step 2: Preferences

* Cities (Dhaka, Chittagong, etc.)
* Budget range
* Campaign types

### Step 3: Dashboard

* Search billboards
* Filter (location, size, price)
* Save shortlist

### Step 4: Booking Flow

* Request availability
* Chat with owner
* Confirm booking

👉 MVP tip:
Start with **manual confirmation (WhatsApp integration)**

---

## 🏗️ B. Billboard Owner Onboarding

### Step 1: Basic Info

* Company name
* Contact person
* Phone

### Step 2: Add Billboard

* Location (Google map pin)
* Size
* Type (LED / static)
* Price

### Step 3: Upload

* Image
* Visibility description

### Step 4: Availability

* Available / booked

### Step 5: Lead Handling

* Receive booking request
* Accept / reject

You can manually update availability at first
